# PHP Code Analysis

## Description

PHP code analyzer for translation to the new language version

## Usage

1. We put the project in the directory next to the repository being checked
2. Copy search.example.csv to search.csv
3. We fill search.csv with a list of functions that are deprecated in the new version of the language.
4. We launch the program with the command: ``` go run main.go ```
5. In the terminal, enter the name of the directory with the repository
6. The utility will display a list of files and the number of errors in the terminal.
